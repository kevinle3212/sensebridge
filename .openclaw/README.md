# OpenClaw — SenseBridge

OpenClaw (and any other AGENTS.md-compatible agent) reads root
[`AGENTS.md`](../AGENTS.md) directly; current repo state lives in
[`AGENT-CONTEXT.md`](../AGENT-CONTEXT.md). Shared, agent-agnostic skills live
in [`.agents/skills/`](../.agents/skills/) — reference them from there rather
than copying into this directory.
